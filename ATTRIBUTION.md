# Attribution

This Neovim configuration was initially seeded from the public Neovim configuration in:

- josean-dev/dev-environment-files

The original structure and plugin layout provided a useful starting point. This repository is maintained independently as `nvim-mrcee` and has been adapted for this environment, including:

- renamed Lua namespace from `josean` to `mrcee`
- OSC52 clipboard behaviour for SSH/Linux/Synology sessions
- local macOS clipboard preservation
- Codex terminal workflow helpers
- local project handover tooling

No private configuration, personal secrets, tokens, keys, or machine-specific account details should be committed to this repository.
